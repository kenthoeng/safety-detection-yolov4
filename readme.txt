//server side
1. "pipenv install --python 3.11" #masukkan di dalam terminal dan pada foler server_side

2. kemudian install requirements.txt, dengan memasukkan perintah "pipenv install -r requirements.txt"

3. untuk memastikan, masukkan perintah "pipenv install python-dotenv"

4. masukkan perintah untuk mengaktifkan enviorements "pipenv shell"

5. "python -m django --version" untuk melihat versi django

6. pindah ke folder "ilham" untuk membuat project django dengan perintah "django-admin startproject wd_ss"

7. maka akan terbuat folder "wd_ss", kemudian rubah nama folder tersebut menjadi "Server Side"

8. masukkan perintah "python manage.py runserver" untuk mengaktifkan website dalam localhost
